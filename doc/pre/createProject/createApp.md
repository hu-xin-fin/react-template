# 创建项目
## 创建项目
`npx create-react-app react-template`
## 启动命令
`npm start`
## 推送到远端
1. 在远端创建项目
    * 主页左边 `repository` 点击 `new` 按钮就可以创建心新项目
2. 把本地的推送到远端
    * 复制 http 地址
    * `git remote add origin https://github.com/hu-xin-fin/react-template.git`
    * `git push -u origin master`
