react-template
====
###### 环境依赖
1. node v16.18.0
2. npm v8.19.2
###### 软件定位&基本功能
* 自我学习的app，主要用于实现基本的表单、列表、以及个人博客功能
* 实践 react 官网的例子
###### 安装环境
`npx create-react-app react-template`
###### 启动命令
`npm start`
